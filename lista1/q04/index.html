Os estilos inline são os que são usados dentro da tag do elemento.

Este elemento foi estilizado usando estilo inline!

O estilo embarcado, por sua vez, fica dentro do head e pode ser aplicado a todos os elementos de uma mesma tag.

Este elemento foi estilizado usando estilo embarcado!

O estilo externo funciona da mesma forma que o embarcado, porém em um arquivo externo, o que facilita o versionamento, modularização e manutenção do código.
